### Project Name
   - 개인 사유 주차장 공유 서비스 PPSS (Private Parking lot Sharing Service)
   
### Purpose
   - 졸업 작품 제출

### Subject
   - 개인 사유 주차장을 공유하고 다른 사용자가 이용할 수 있는 애플리케이션 개발
   
### Main Function
  - 등록 서비스 (인증 , 주차 공간 등록 , 관리)
  - 사용 서비스 (인증 , 주차 공간 사용 , 검색 및 찾기)
  - 구글 지도를 이용한 등록된 주차장, 남은 공간 찾기, 내 주변 주차장 찾기
